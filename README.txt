


Esta aplicación fue desarrollada como parte de un examen para TACTICASOFT continuación, se proporciona información útil para compilar y ejecutar la aplicación.

Instrucciones de Uso:

1. Abre el proyecto en Visual Studio.
2. Compila la solución.
3. Ejecuta la aplicación.


Dependencias:

- FontAwesome.
- Log4net.
- Stimulsoft.

Base de Datos:

- Base de datos SQL Server (brindada por tacticasoft).


Información del Proyecto:

- El proyecto es una app e-commerce usando el patron MVVM (models-view-viewModels).

- Las vistas se crearon en WPF.

- Se creo en .NET (C#).

- Se uso Visual Studio 2019 como entorno de desarrollo.
